# Security policy

Codex skills can influence tool use and may include executable helpers. Please
report prompt-injection paths, unsafe default actions, secret exposure, or
script vulnerabilities through GitHub's private security advisory flow rather
than a public issue.

Include the affected skill, a minimal reproduction, expected behavior, and the
potential impact. Do not include real credentials, private workspace data, or
unpublished material in the report.

The project does not publish a fixed support window. Security fixes target the
current `main` branch.
